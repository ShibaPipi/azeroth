## 小数的补码

>### 小数补码的定义
* 设一数字 x，n 为 x 的位数
    * 若 0 ≤ x < 1，则 x 的值为 x 本身
    * 若 -1 ≤ x < 0，则 x 的值为 2 + x
* 整数的规律同样适用于小数，即
    * 负数的反码等于原码除符号位外按位取反
    * 负数的补码等于反码 + 1
* 举例
    * 设 n = 4，x = 9 / 16，计算 x 的二进制原码，反码和补码
        * 原码：0,0.1001
        * 反码：0,0.1001
        * 补码：0,0.1001
    * 设 n = 4，x = -11 / 32，计算 x 的二进制原码，反码和补码
* 原码：1,0.01011
* 反码：1,0.10100
* 补码：1,0.10101
